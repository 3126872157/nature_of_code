### 第七章 元胞自动机
#### 定义
+ 由**细胞**对象组成的系统模型（三要素：网格，状态，领域）
  + 细胞存在于网格上（可以扩充到任意有限维度的空间中）
  + 细胞有有限的状态（最简单的是生死，即 0 和 1）
  + 每个细胞都有一个领域（可以有多种方式定义领域，通常指与该细胞相邻的所有细胞）

#### 基本的元胞自动机
+ 细胞状态转移函数：下一时刻的状态是上一时刻该细胞领域的状态的函数值
+ 而这个转移函数，实际上即是**规则集**，想想图像处理中的腐蚀膨胀运算，也是 CA 的思想哦
+ 但是一般来说 CA 的规则集可以不遵循算数逻辑，可以只是简单的**输入和输出的任意映射**
+ 题外话：一维元胞自动机其中一条规则集可以生成谢尔宾斯基三角，太tm壮观了
